# 《同谋》· 重要情节

---

## 一、财富发现线 · 扮猪吃老虎的逐步揭开

> 这不是独立的故事线——是埋在主线里的一串地雷，每隔一段路就炸一颗，炸一次就让苏棠（以及后来加入的苏念）重新认识一次自己的丈夫/姐夫。

### 苏棠的发现弧线

```
【第一阶段 · 小裂痕（婚后半年到一年）】
他偶尔在超市不看价格签 → 她看了一眼他的工资条 → 不够他花的
→ 她没问。不是不敢，是不知道怎么开口

【第二阶段 · 第一次冲击（婚后第一年底）】
瞥到他的美股账户App → 以为几千美金试试
→ 某天他洗澡时电脑没关 → 七位数，美元
→ 她在那张椅子上坐了十分钟
→ 她说："这个……是真的？"
→ 他说："嗯。"
→ 她没追问，但那天晚上没怎么睡着

【第三阶段 · 全球化暴露（婚后第二年）】
日文邮件弹窗 → "大阪"+"予約"
→ "你在大阪有房子？" → "两套。做民宿用的。"
→ 发现海外账户文件夹 → 数字让她合上了电脑
→ 那天晚上她看着对面穿旧T恤的他
→ 觉得自己好像从来没真正认识过他

【第四阶段 · 摊牌与接纳】
"你到底还有多少事没告诉我？"
→ 他从头讲了一部分 → 精确的投资轨迹、海外资产
→ 她听懂了那个总数
→ 她问："那你为什么还每天去上那个班？"
→ 他愣了一下："因为不上班的话，白天也不知道干什么。"
→ 聊到凌晨三点——她第一次触碰到他普通衬衫底下的那部分人生

【后续 · 持续的冲击】
他会主动跟她讲这周进了什么仓位
→ 但隔一段时间还是会被震到：
  · 发现他在东京和大阪不止两套，是四套
  · 发现他大学时借父母的钱后来还的是一套别墅的首付
  · 发现每一次大的跃升都踩在历史事件的节点上
```

### 苏念的发现弧线

```
第一次 · 无意瞥见：借用姐夫书房，看到手写英文数字纸——最后面有好几个逗号
第二次 · 姐姐说漏嘴："你姐夫最近又折腾了点投资"——苏棠说这话时的表情很复杂
第三次 · 亲眼看到：海外银行信封露出一角——数字格式和她的兼职工资不一样
第四次 · 姐妹坦诚后：苏棠告诉她真相 → 苏念沉默 → 对姐夫的所有想象都不对了
```

### 知情圈递进

1. 最早：只有陈屿自己知道
2. 婚后逐步：苏棠发现并知情
3. 中期：苏念加入知情圈
4. 极后期（三人汇合后）：沈鹿溪进入核心圈后被告知
5. 永不开放：林晚棠、方柠等外围女性

## 二、情感线关键转折点（待细化）

### 苏念线转折

- 大学第一次撞破姐姐姐夫宿舍性爱（种子深埋）
- 苏棠第一次在家庭聚餐中"无意"安排苏念坐陈屿旁边
- 苏棠第一次在三人独处时找借口离开（"我去厨房看看汤"）
- 苏念第一次察觉姐夫的目光在自己身上停得比以往久
- **苏念暂住姐姐家后，在主卧自慰被苏棠撞见（两人沉默，未戳破——但关系已变）**
- 饭桌下第一次用脚碰姐夫小腿（试探未被拒绝，逐月升级）
- **苏棠提出"直播"条件**
- 第一次直播观看
- **苏念第一次发生（日本吻后，通过直播完成。次日敬茶认序：跪地端茶喊"姐姐喝了这杯茶"）**
- **夫妻谈心：陈屿坦承性欲异于常人，苏棠接受。两人达成"不会瞒她、不会拦他"的默契**
- 苏念发现姐姐的"不阻拦"
- 姐妹摊牌与和解
- 和解后第一次共同参与

### 沈鹿溪线转折

- **陈屿暗中解决沈鹿溪家弟弟学费问题。他对她说"你不是缺钱，你是缺人帮你"——她从此接受做小也没关系**
- **苏棠对沈鹿溪摊牌："你喜欢他吧。我早就知道了。"（沈鹿溪崩溃→苏棠握着她手说"我不介意"）**
- 苏棠对陈屿说出"鹿溪周末来家里吃饭"——共谋的开始
- 第一次留陈屿和沈鹿溪独处
- 沈鹿溪酒后的眼神变化
- **苏棠辅助沈鹿溪完成第一次（帮她脱衣、抱着她、扶着她的腰打开腿）**
- 陈屿事后从背后抱住苏棠（无声的确认）
- 沈鹿溪进入知情圈

### 共同转折

- 苏念与沈鹿溪彼此知道对方的存在
- 三人汇合（大被同眠）
- 四人同场（终极开放形态）
